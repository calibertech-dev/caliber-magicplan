# Caliber Salesforce x magicplan

The Salesforce x magicplan integration by Caliber Technologies adds several useful functions to make using them together easier.

## Features:
-  Adds Magicplan_Project__c custom object to link to any other Salesforce records.
-  Create new magicplan project from within Salesforce
-  Get Reports from magicplan and create a new ContentVersion in Salesforce
-  Search remote magicplan projects from Salesforce
-  Create deeplink and web link to your magicplan project from Salesforce
-  Webhook Handler in Salesforce: useful for automatically downloading new changes from magicplan
